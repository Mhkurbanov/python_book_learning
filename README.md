## Higher or Lower

### Описание
Учебная консольная игра на Python.

Игроку предлагается угадать, будет ли следующая карта выше или ниже текущей.
Проект написан в процессе изучения Python и работы с Git/GitHub.

### Чему посвещен проект
- практика базового Python(условия, циклы, функции)
- работа со списками и словарями
- основы алгоритмического мышления
- практика Git и GitHub

### Правила игры 
- Игра начинается с 50 очков
- За правильный ответ начисляются +20 очков
- За неправильный ответ снимаются -15 очков
- Игра длится несколько раундов

### Запуск 
```bash
python HigherOrLower.py
